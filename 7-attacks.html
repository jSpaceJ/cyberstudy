Vulnerabilities & Attacks

Types of Attacks

  • Passive: Eavesdropping, traffic analysis.
  • Active: Modification, replay, masquerade, denial of service.

Examples

  • Replay Attack: Capturing and reusing a valid message.
  • Masquerade: Pretending to be another user.
  • Timing Attack: Infers secrets from runtime differences.
  • DoS: Flooding a server to exhaust resources.

Vulnerabilities

  • Design-level: Protocol flaw (e.g., always using same prime in RSA).
  • Implementation-level: Side-channel leaks (timing, memory errors).
